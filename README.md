# 🐶 AI-DOGTOR  
**Intelligent Virtual Assistant for First Aid**

DOGTOR is a virtual assistant designed to **detect and correct errors from other assistants**, providing reliable, clear, and useful information in critical situations. It specializes in **first aid**, delivering fast and accessible guidance through an integrated web platform.

---

# 🚀 Project Overview

DOGTOR aims to improve the quality of responses in sensitive contexts, especially emergencies. It not only acts as an assistant, but also as an **intelligent verifier**, capable of identifying inconsistencies and offering safer recommendations.

The web platform complements the assistant with essential tools, allowing users to quickly access medical and emergency resources in real time.

---

## 🧠 Key Features

- ✅ **Correction of other assistants' responses**
- 🩺 **First aid specialization**
- 📍 **Nearby hospital locator**
- 🎥 **Educational video library**
- 📞 **Emergency call button**
- 💬 **User-friendly conversational interface**
- ⚡ **Fast and optimized responses**
- 🔒 **Focus on safety and reliability**

---

## 🌐 Web Features

The web application includes:

- Direct access to the virtual assistant  
- Interactive map with nearby hospitals  
- Educational content (videos and guides)  
- Emergency contact button  
- Responsive and intuitive interface

## ⚙️ Installation & Usage

1. Clone the repository:
   ```bash
   git clone
2. Install dependencies:
  ```bash
  cd dogtor
  npm install
  ````
3. Run the project:
  ```bash
  npm run dev
  ````
4. Open in your browser:
  ```bash
  http://localhost:3000
  ````
5. Use it
